# INSTRUÇÕES

  1. Baixar o arquivo Gnomo.jar
  2. Pressione a tecla Windows + R
  3. digite na caixa de texto: %appdata%
  4. vá até a pasta .minecraft
  5. dentro da pasta .minecraft procure a pasta mods
  6. se não houver a pasta mods crie uma pasta chamada mods
  7. dentro dessa pasta coloque o arquivo Gnomo.jar
  8. Você precisará cumprir alguns requisitos que estão em Dependências.md
